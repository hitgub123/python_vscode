## 基本設計書

| ID      | fun-028                      |
| ------- | ---------------------------- |
| 分類    | 勤怠データの分析・レポート               |
| 機能名  | チーム勤怠サマリーダッシュボード機能                   |
| 概要    | 管理職が自身のチームの遅刻、早退、欠勤、時間外労働などの勤怠状況を一覧で確認できるインタラクティブなダッシュボード。 |
| 利用者  | 管理職                   |
| 作成者  |                               |
| 作成日付 |                               |

### 1. 目的

本機能は、管理職が自身のチームの勤怠状況を一覧で確認できるダッシュボードを提供することを目的とする。

### 2. 適用範囲

本機能は、管理職がチームの勤怠状況を把握する必要がある場合に適用される。

### 3. 入力

#### 入力項目

| 項目       | 内容                  | 型       | 必須   | 備考                                                                                       |
| ---------- | --------------------- | -------- | ------ | ------------------------------------------------------------------------------------------ |
| 管理職ID   | 管理職を識別するID     | 文字列   | 必須   | 社員番号など                                                                               |
| 対象期間   | 勤怠状況を確認する期間 | 日付型   | 必須   | 開始日と終了日                                                                           |
| 部門/チーム選択 | 勤怠状況を確認する部門またはチーム | 文字列   | 任意   | 指定しない場合は、管理職が管轄する全ての部門/チームを表示                                                                        |

#### 業務Check

| 項目       | 内容                                                                                   |
| ---------- | ------------------------------------------------------------------------------------ |
| 管理職ID   | 社員マスタに登録されている管理職IDであること。                                                              |
| 対象期間   | 開始日は終了日以前であること。                                                            |
| 部門/チーム選択   | 選択された部門/チームが、管理職が管轄する部門/チームのいずれかであること。存在しない場合はエラー。                               |
